# Лендинг для wellness-продуктов

## Обзор проекта
Современный лендинг с премиальными анимированными фоновыми эффектами и природной цветовой палитрой. Цель лендинга — мотивировать посетителей перейти на основной сайт каталога. Проект создан на React + Vite + TypeScript + Tailwind CSS + Framer Motion.

## Последние изменения (3 декабря 2025)

### 10 вариантов Hero блока
Полностью переработанный Hero с системой переключения вариантов:

**Доступные варианты:**
1. **Классический Split** - две колонки 50/50, текст слева, изображение справа
2. **Полноэкранный Overlay** - fullscreen изображение с тёмным overlay, текст по центру
3. **Ассиметричный** - 60/40 разбивка с декоративными кругами
4. **Карточный Hero** - центрированная карточка поверх паттерна
5. **Минималистичный** - много пустого пространства, крупный заголовок
6. **Видео Placeholder** - готов для замены на видео
7. **Сетка продуктов** - 3 мини-превью продуктов с hover эффектами
8. **Диагональный Split** - clip-path диагональное разделение
9. **Стековый/Layered** - многослойный дизайн с floating элементами
10. **Интерактивный Slider** - автоматическая карусель с 3 слайдами

**Механизм переключения:**
- Floating selector в правом верхнем углу (fixed position)
- Мгновенное переключение между вариантами
- LocalStorage сохранение выбранного варианта
- Анимированные переходы через Framer Motion

**Файлы:**
- `src/components/HeroVariants.tsx` - все 10 вариантов + селектор
- Интеграция в `src/pages/Home.tsx`

### Премиальная система фоновых эффектов
Полностью переработанные анимированные фоны взамен простых вариантов:

**1. Анимированный Mesh-градиент**
- Плавно переливающиеся радиальные градиенты
- 20-секундный цикл анимации
- Создаёт "живой" эффект фона

**2. Дышащие Blob-формы**
- Три органические blob-формы с разными размерами
- Анимация масштаба и изменения формы (border-radius)
- Размытие для мягкого эффекта

**3. Плавающие частицы**
- 25 частиц со стабильными позициями (детерминированное размещение)
- Лёгкое движение вверх-вниз и влево-вправо
- Opacity-анимация для "дыхания"

**4. Параллакс-ботаника**
- SVG-силуэты листьев на разных слоях
- Движение с разной скоростью при скролле
- Лёгкое вращение для динамики

**5. Анимированные волновые разделители**
- WaveDivider - между секциями
- OrganicWaveDivider - органические волны
- Плавное изменение формы волн

**6. Текстура шума**
- Очень subtle (0.015 opacity) текстура поверх
- Добавляет "материальность"

### Accessibility: Reduced Motion
Полная поддержка `prefers-reduced-motion`:
- Custom hook `useReducedMotion` для определения настроек
- Все анимированные компоненты принимают `reducedMotion` prop
- При включённом reduced motion показываются статичные версии

## Структура продуктов
6 разных товаров (не все пластыри):
1. **Пластырь** - для точечного воздействия
2. **Очиститель воды** - структурирует и очищает воду
3. **Очелье** - налобная повязка для головы
4. **Наочники** - для глаз и зрения
5. **Кушак** - пояс для спины и живота
6. **Тесьма** - для рук, ног и суставов

## Секции лендинга
- **Hero** - главный баннер с CTA
- **Как это работает** - 3 шага с иллюстрацией
- **Каталог продуктов** - зигзаг layout с карточками (зеленоватый фон)
- **Отзывы клиентов** - 4 отзыва с аватарами
- **FAQ** - 5 вопросов-ответов с аккордеоном
- **CTA** - финальный призыв к действию

## Архитектура проекта

### Структура файлов
```
/
├── src/
│   ├── pages/
│   │   └── DesignVariants.tsx  # Главная страница с эффектами
│   ├── App.tsx                 # Роутинг приложения
│   ├── main.tsx                # Точка входа React
│   └── index.css               # Глобальные стили
├── index.html                  # HTML шаблон
├── vite.config.ts              # Конфигурация Vite
├── package.json                # Зависимости проекта
└── tsconfig.json               # Настройки TypeScript
```

### Компоненты фоновых эффектов (в DesignVariants.tsx)
- `useReducedMotion()` - hook для определения reduced motion
- `AnimatedMeshGradient` - mesh-градиент с анимацией
- `BreathingBlobs` - "дышащие" blob-формы
- `FloatingParticles` - плавающие частицы (используют STABLE_PARTICLES)
- `ParallaxBotanicals` - параллакс SVG-листья
- `WaveDivider` / `OrganicWaveDivider` - волновые разделители
- `SubtleTexture` - overlay текстура шума

### Технологический стек
- **React 19** - UI библиотека
- **TypeScript** - типобезопасность
- **Vite** - быстрый сборщик и dev сервер
- **Tailwind CSS v4** - utility-first CSS фреймворк
- **Framer Motion** - анимации, useScroll, useTransform
- **Wouter** - лёгкий роутинг
- **Lucide React** - иконки

### Конфигурация
- **Dev сервер**: порт 5000, host 0.0.0.0
- **Production build**: статический сайт в папке `dist/`
- **HMR**: настроен для работы через WebSocket Secure (wss://)

## Команды
- `npm run dev` - запуск dev сервера (порт 5000)
- `npm run build` - сборка для production
- `npm run preview` - предпросмотр production сборки
