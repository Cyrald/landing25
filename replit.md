# Лендинг для wellness-продуктов

## Обзор проекта
Минималистичный лендинг для wellness-продуктов с чистым дизайном. Цель — мотивировать посетителей перейти на основной сайт каталога. Проект создан на React + Vite + TypeScript + Tailwind CSS + Framer Motion.

**Дизайн-философия**: Экстремальный минимализм. Секции разделяются только цветом фона — никаких декоративных элементов.

## Последние изменения (4 декабря 2025)

### Минималистичный редизайн
Полная очистка от декоративных эффектов:
- **Удалены все фоновые эффекты**: AnimatedMeshGradient, BreathingBlobs, FloatingParticles, SubtleTexture
- **Удалены все разделители**: WaveDivider, OrganicWaveDivider
- **Удалён useReducedMotion hook** (больше не нужен)
- Секции разделяются только разным цветом фона

### Hero блок "История Исцеления"
Единственный Hero блок с эмоциональным повествованием:
- Цитата реального клиента с историей успеха
- Заголовок "Тысячи историй исцеления"
- Кнопка "Читать истории"
- Изображение продукта справа

**Файлы:**
- `src/components/HeroVariants.tsx` - Hero блок
- `src/pages/DesignVariants.tsx` - Главная страница (чистый минимализм)

## Структура продуктов
6 разных товаров:
1. **Пластырь** - для точечного воздействия
2. **Очиститель воды** - структурирует и очищает воду
3. **Очелье** - налобная повязка для головы
4. **Наочники** - для глаз и зрения
5. **Кушак** - пояс для спины и живота
6. **Тесьма** - для рук, ног и суставов

## Секции лендинга
- **Hero** - главный баннер с историей исцеления и CTA
- **Как это работает** - 3 шага с иллюстрацией
- **Каталог продуктов** - зигзаг layout с карточками (зеленоватый фон)
- **Отзывы клиентов** - 4 отзыва с аватарами
- **FAQ** - 5 вопросов-ответов с аккордеоном
- **CTA** - финальный призыв к действию

## Разделение секций по цвету фона
```
Hero        → bg (#f8faf8)
How It Works → bg (#f8faf8)
Products    → catalogBg (#f4f9f5) - зеленоватый
Testimonials → bg (#f8faf8)
FAQ         → bgAlt (#eef4ee) - чуть темнее
CTA         → gradient (зелёный градиент)
```

### Технологический стек
- **React 19** - UI библиотека
- **TypeScript** - типобезопасность
- **Vite** - быстрый сборщик и dev сервер
- **Tailwind CSS v4** - utility-first CSS фреймворк
- **Framer Motion** - анимации появления при скролле
- **Wouter** - лёгкий роутинг
- **Lucide React** - иконки

### Конфигурация
- **Dev сервер**: порт 5000, host 0.0.0.0
- **Production build**: статический сайт в папке `dist/`
- **HMR**: настроен для работы через WebSocket Secure (wss://)

## Команды
- `npm run dev` - запуск dev сервера (порт 5000)
- `npm run build` - сборка для production
- `npm run preview` - предпросмотр production сборки
